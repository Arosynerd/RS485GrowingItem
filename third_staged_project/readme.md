## 功能

主机发送从机，每隔一段时间询问从机温湿度及光照，从机应答。主机再发送数据到tcpserver

tcpserver保存数据到数据库。若没有连接到tcpserver，暂时保存在主机的外部存储器，连接后再发送给tcpserver。

主机从机都有一个频闪指示灯代表正常运行。

目前从机拓展不灵活

## 逻辑过程

主机每隔5秒向从机们询问数据、主机将获得的数据整合在一起发送给tcpserver。

tcpserver实时显示数据，接收到的数据存在数据库中，可以查询历史数据。
